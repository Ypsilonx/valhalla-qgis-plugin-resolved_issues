about=Valhalla provides access to most of the functions of the open-source Valhalla routing engine. The tool set includes routing, isochrones and matrix calculations, either interactive in the map canvas or from point files within the processing framework. Extensive attributes are set for output files, incl. duration, length and start/end locations.
